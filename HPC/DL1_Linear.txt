3. Linear Regression Theory
Equation:
y=w1x1+w2x2+...+b
Idea:
Find a straight line that best fits data.

Advantages:
Simple
Fast
Interpretable
Disadvantages:
Cannot capture complex patterns
Assumes linear relationship

4. Deep Learning Regression
Neural Network idea:
Instead of straight line → learns complex patterns
Structure:
Input → Hidden layers → Output
Each neuron does: y=activation(wX+b)
Why multiple layers?
First layer → simple patterns
Middle layers → complex patterns
Last layer → final prediction

5. Activation Function (ReLU)
f(x)=max(0,x)
Why used?
Removes negative values
Helps network learn non-linear patterns
